## 1. Responsive Breakpoints

- Compact / Mobile (`< 600dp`): Single-column, bottom navigation bar.
- Medium / Tablet (`600dp - 1024dp`): Two-column split view, navigation rail.
- Expanded / Desktop (`> 1024dp`): Three-column layout, persistent navigation sidebar, keyboard shortcuts.
